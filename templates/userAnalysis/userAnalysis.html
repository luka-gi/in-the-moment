Features


  • User Creation

    Create a user and password.

  • Login

    Allow users to login. That login is authenticated.

  • Edit User

    Change user information, such as a password.

  • Reminders/Generic Pin - Address Input

    Create geolocation-based reminders/pins by typing in an address.

  • Reminders/Generic Pin - Interactive Map

    Create geolocation-based reminders/pins by dropping a pin on the map.

  • Generate Route

    Choose which pins to include and generate a route through selected.

  • Edit Route

    Customize the path you take to reach your pins.

  • Edit Reminders/Pins

    Change the information in the reminder, such as location, message, or time.

  • Delete Reminders/Pins

    Remove reminder from your list.

  • Display Reminders/Pins

    Show all existing reminders

  • Display Nearby Reminders/Pins

    Show all reminders in a given radius

  • Images in Reminders/Pins

    Embed images for enhanced reminders.

  • Pause Reminders/Pins

    Sleep reminders indefinitely, on a schedule, or for a given period of time.

  • Bundle Reminders/Pins

    Select and group pins or reminders.

  • Share Reminders/Pins

    Share individual or grouped reminders.
